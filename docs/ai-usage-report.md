- Faster development
- Better understanding of JavaScript concepts

- Improved UI design ideas

- Adjusted structure to fit assignment requirements

## Risks of Using AI
- Over-reliance may reduce understanding
- AI can produce incorrect or inefficient code
- Code may not match assignment requirements

## How I Reduced Risks

- Tested every feature manually
- Simplified code when needed
- Ensured I understand each part
